### Project Prerequisites
* Lombok Plugin for IntelliJ
    * Enable "Annotation Processing" in IntelliJ
* Have `git` installed;
    * macOS;
        * https://git-scm.com/download/mac
        * `brew install git` (only if you have homebrew installed)
    * Windows
        * https://git-scm.com/download/win
